This plugin locks down front end read access to all post types.  Creates an anonimous role, if one doesn't already exist;

== Description ==

This plugin requires a separate plugin for modification of role capabilities.

== Installation ==

1. Upload `sk_post_type_access` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
